Serialport-server

access SerialPort via HTTP, WebSocket and TCP Socket

Download as .zip Download as .tar.gz View on GitHub

INSTALL:

  • gem install serialport-server

SYNOPSIS:

  • % serialport-server --help
  • % serialport-server /dev/tty.usb-device
=> start HTTP, WebSocket, TCP Socket servers.

HTTP Interface:

read serialport
  • % curl http://localhost:8783
write
  • % curl -d 'hello!!' http://localhost:8783

TCP Socket Interface:

  • % telnet localhost:8784

WebSocket Interface:

  • var ws = new WebSocket("ws://localhost:8785");
