= serialport-server

  • http://github.com/shokai/serialport-server

== DESCRIPTION:

access SerialPort via HTTP, WebSocket and TCP Socket.

== SYNOPSIS:

% serialport-server --help % serialport-server /dev/tty.usb-device -http_port 8783 -websocket_prot 8784 -socket_port 8785

== REQUIREMENTS:

  • USB Device (Arduino, mbed ...)

== INSTALL:

  • gem install serialport-server
